# AGENTS.md

本專案是「國豐當舖」全新官網，與宥達宸網站完全分開維護。

## 專案定位

- 品牌名稱：國豐當舖
- 地址：台北市大同區民族西路78號1樓
- 統一編號：15092702
- 視覺方向：還原舊資料中的「藍色元素版」，使用淺藍、深藍、黃色點綴與插圖風格。

## 技術規範

- 使用 Next.js App Router。
- 使用 TypeScript。
- 使用 Tailwind CSS。
- 使用 pnpm。
- 新頁面放在 `app/`，共用元件放在 `components/`，內容資料放在 `data/`。

## 合規與文案

- 不得保證核貸、不得使用「保證過件」「百分百核准」「免審核」等高風險字眼。
- 當舖服務需使用保守說法，借款條件、利息、費用與額度以現場評估、契約及法令規範為準。
- 不得誇大「超低利」「一定撥款」「立即無條件借款」。
- 若沿用舊素材中的舊品牌或舊地址，必須改為國豐當舖目前資料。

## 品牌隔離

- 不要引用宥達宸網站的資料、文案、Logo、API 或部署設定。
- 本專案未來應獨立 Git repo、獨立 Vercel project、獨立網域與環境變數。
