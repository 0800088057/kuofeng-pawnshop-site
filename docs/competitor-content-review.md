# 國豐當舖同業內容補強方向

本文件整理同業當舖網站常見內容結構，並轉換為國豐當舖可落地的網站補強項目。

## 同業常見做法

1. 服務項目切分清楚
   - 多數網站會把汽車借款、房屋二胎、支票貼現、工商融資、代償等拆成獨立頁。
   - 國豐已具備服務分頁，接下來重點是補足每頁的情境、資料與注意事項。

2. CTA 明顯
   - 同業通常會放電話、LINE、線上諮詢。
   - 國豐保留電話、LINE 與表單，但文案避免保證式或過度強銷。

3. 應備文件與流程
   - 使用者最常在意「要帶什麼」「多久知道結果」「是否一定可以借」。
   - 國豐新增首頁、流程頁與 FAQ 的文件清單與評估提醒。

4. 合法與透明說明
   - 合法當舖應清楚說明實際條件以現場評估與契約為準。
   - 國豐補強「不保證承作、不保證額度、費用與利息簽約前說明」。

5. 長尾 SEO 問答
   - 汽車還有貸款能不能問、房屋二胎要什麼資料、支票貼現多久知道結果，都是適合補成 FAQ 的長尾問題。
   - 國豐已新增 FAQ，後續可再依真實來電問題擴充。

## 已落地調整

- 導覽新增服務項目、流程、常見問題、聯絡我們。
- 首頁新增「先確認需求，再選適合的服務」內容區。
- 流程頁新增到店前資料準備與注意事項。
- FAQ 擴充汽車借款、房屋二胎、支票貼現、資料保密、網站資訊限制等問題。
- 各服務分頁新增「適合先諮詢的情境」與「評估前提醒」。

## 後續建議

1. 依真實客服問題，持續擴充 FAQ。
2. 每個服務頁可再加入 600-1000 字 SEO 內容，但要避免保證借款、保證核准等高風險字眼。
3. 加入 Google 商家資訊、實體門市照片與交通資訊，可再強化在地信任感。
4. 若有合法登記證照或門市照片，可放到關於/聯絡頁，增加轉換。
